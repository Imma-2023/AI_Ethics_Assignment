
## Project Description

This assignment explores ethical AI principles, case studies on bias (Amazon hiring tool, facial recognition), and practical audits of the COMPAS recidivism dataset using AI Fairness 360 (AIF360).

## Instructions

1. **Jupyter Notebook:** Open `notebooks/COMPAS_Fairness_Audit.ipynb` to run the bias audit.  
2. **Python Script:** Run `src/fairness_audit.py` (optional) to execute audit scripts.  
3. **Requirements:**  
   - Python 3.x  
   - Libraries: `aif360`, `pandas`, `matplotlib`, `scikit-learn`  
   - Install via: `pip install aif360 pandas matplotlib scikit-learn`
